# Product Requirements Document (PRD)

## Product Name
Personal Portfolio Website – Data Analyst

## Owner
Shubham Shirsekar

## Purpose & Vision
The purpose of this website is to position Shubham Shirsekar as a strong, business-oriented Data Analyst with real-world experience in analytics, automation, and API integrations. The site should clearly communicate technical depth, business impact, and project-based problem-solving to recruiters and hiring managers.

The website must:
- Showcase projects as case studies, not just repositories
- Be recruiter-friendly, fast, and SEO-optimized
- Signal professionalism, clarity, and analytical thinking

Primary goal: Secure interviews for **Data Analyst / Business Analytics internships (May 2026)**.
