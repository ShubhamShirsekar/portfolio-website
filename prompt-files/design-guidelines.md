# Design Guidelines

## Design Philosophy
- Clean
- Minimal
- Data-first
- Recruiter-friendly

## Typography
- Primary Font: Inter
- Headings: Semibold / Bold
- Body: Regular

## Colors
- Primary: Neutral Dark (Gray-900)
- Accent: Blue / Indigo
- Background: White / Gray-50
- Success: Green
- Warning: Amber

## Spacing
- Use Tailwind spacing scale only
- Section padding: py-16 / py-24
- Card gap: gap-6 / gap-8

## Border Radius
- Cards: rounded-xl
- Buttons: rounded-lg

## Shadows
- Subtle only (shadow-sm / shadow-md)

## Animations
- Framer Motion
- Use only for:
  - Page transitions
  - Card hover
  - Section fade-in
- Avoid excessive motion
