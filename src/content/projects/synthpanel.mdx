## Overview

Synthpanel is an interactive AI chatbot application built using Python and Streamlit. It provides users with a seamless chat interface to interact with language models, featuring a responsive design and straightforward user experience.

---

## Features

- **Interactive Chat Interface**: Built entirely with Streamlit, offering real-time responses.
- **Easy Deployment**: Accessible globally via Streamlit Community Cloud.
- **Responsive Design**: Works well across different devices and screen sizes.
