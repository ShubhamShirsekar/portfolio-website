## Overview

End-to-end ML and LLM pipeline on Azure OpenAI and Streamlit that clusters 2,400+ Spanish consumer finance survey records into validated customer personas, instantiates them as interactive synthetic panellists with two-layer response validation and confidence scoring, and surfaces product-persona fit scores via an analyst-facing heatmap dashboard.

---

## Features

- **Interactive Chat Interface**: Built entirely with Streamlit, offering real-time responses.
- **Easy Deployment**: Accessible globally via Streamlit Community Cloud.
- **Responsive Design**: Works well across different devices and screen sizes.
